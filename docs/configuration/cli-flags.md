# CLI Flags

| Flag                            | Description                            | Required | Default |
| ------------------------------- | -------------------------------------- | -------- | ------- |
| `-c, --create-extension <name>` | name of extension                      | Yes      | N/A     |
| `-t, --template [type]`         | name template                          | No       | starter |
| `-i, --install`                 | if provided, will install qext-scripts | No       | N/A     |
